En  esta etapa de desarrollo, el alumno realizará un sitio web con diferentes paginas HTML, donde se identifiquen y utilicen las diferentes etiquetas.

En esta práctica solo se debe agregar el contenido, no es necesario darle estilo, ni funcionalidad (esto se hará en futuras prácticas).
Crea un archivo index.html principal agrega un formulario para crear una página de login con username y password.
Crea otro archivo tags.html donde utilices diferentes etiquetas HTML.
Crea un archivo profile.html donde crees una pagina de perfil, agrega tus datos y una foto.
Agrega links en cada archivo html para tener navegación entre las diferentes paginas.
Como opcional puedes crear más archivos HTML y añadir más imágenes.
Recuerda utilizar las siguientes etiquetas:
<!DOCTYPE>
<html>
<head>
<body>
<meta>
<title>
<header>
<nav>
<p>
<br />
<a>
<img>
<strong>
<sup>
<sub>
<div>
Crea un repositorio publico en GitHub y agrega este proyecto con todos sus archivos
